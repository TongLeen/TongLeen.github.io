# 安装配置

## 环境要求

以下是使用该库的环境要求：
- Bun >= 1.3.10
- 网络连接（下载库需要）

## 安装步骤

### 安装 Bun

安装过程可以参考 [Bun 官方](https://bun.sh/)

安装完成后，在终端输入
```sh
bun -v
```

得到
```sh
$ bun -v
1.3.12
```

即代表安装成功

### 安装库

进入到工作文件夹
```sh
cd /path/to/your/workspace/
bun add @tongleen/tcad-script-gen
```

控制台输出以下内容表示安装成功
```sh
$ bun add @tongleen/tcad-script-gen
bun add v1.3.12 

installed @tongleen/tcad-script-gen@0.1.0

1 packages installed
```
